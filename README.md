#  Exploratory Data Analysis (EDA) Project

## Overview

This project performs **Exploratory Data Analysis (EDA)** on a dataset using Python.
The goal is to understand the data, identify patterns, handle missing values, and generate meaningful insights.

---

## Tools & Technologies Used

* Python 
* Pandas
* NumPy
* Matplotlib
* Seaborn
* fg-data-profiling

---

## Project Structure

EDA-Task/
│── eda.py
│── EDA.ipynb
│── report.html
│── data/

---

## Steps Performed

1. Data loading and preprocessing
2. Handling missing values
3. Statistical analysis using `describe()`
4. Data visualization (count plots, heatmaps)
5. Correlation analysis
6. Automated report generation using fg-data-profiling

---

## Key Insights

* Dataset contains multiple features with both numerical and categorical data
* Missing values were found in some columns
* Certain features show strong correlations
* Distribution of data varies across different attributes
* Visualizations helped in understanding patterns clearly

---

## Output

* `EDA.ipynb` → Contains analysis and visualizations
* `report.html` → Full automated EDA report

---

## How to Run

1. Install required libraries:

   ```
   pip install pandas numpy matplotlib seaborn fg-data-profiling
   ```
2. Run the script:

   ```
   python eda.py
   ```
3. Open `report.html` in browser

---

**Note:** report.html is large, please download and open locally.


## Conclusion

EDA helps in understanding the dataset before applying machine learning models.
This project demonstrates how to analyze data efficiently using Python tools.
